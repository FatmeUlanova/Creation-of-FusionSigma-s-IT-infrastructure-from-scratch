# FusionSigma – IT Infrastructure and Application Project

This repository contains the final project developed for my IT diploma.  
The project consists of the design and implementation of a complete IT environment including infrastructure, database architecture, and a working web application.

**Important:**  
The full documentation of this project is written in **Spanish**, as it was originally developed for an academic program.

---

# Project Overview

FusionSigma is a comprehensive project that covers several core areas of enterprise IT:

- Network architecture design
- Systems infrastructure planning
- Operating system configuration
- Database design and implementation
- Development of a PHP web application connected to the database

The project simulates the creation of a small enterprise IT environment from scratch.
